# 맥북 숏컷 활용기: 클립보드에서 필요한 링크만 붙여넣기

웹에서 무언가를 복사하다 보면, 내가 필요한 값보다 더 많은 문구가 함께 클립보드에 들어올 때가 있다.

예를 들면 이런 식이다.

```text
아래 링크를 참고하세요.

    https://inf.run/Z6cNw
```

내가 따로 기록해두고 싶은 건 링크 하나뿐인데, 매번 붙여넣은 뒤 앞뒤 문장을 지우는 일이 반복된다.

```text
https://inf.run/Z6cNw
```

한두 번이면 괜찮다.
그런데 메모, 문서, 개인 노트에 자주 정리하다 보면 이 작은 편집이 계속 거슬린다.

여기서 만드는 건 공개 공유용 문구를 바꾸는 도구라기보다, 개인 메모나 작업 노트에 필요한 값만 빠르게 정리하기 위한 클립보드 자동화다.

이럴 때 macOS의 **단축어(Shortcuts)** 앱을 쓰면 꽤 간단하게 해결할 수 있다.
클립보드에 들어 있는 텍스트에서 내가 원하는 URL 패턴만 찾아 다시 클립보드에 넣고, 바로 붙여넣기까지 실행하는 식이다.

## 최종 동작

완성 후 사용 흐름은 다음과 같다.

```text
1. 웹페이지에서 텍스트를 복사한다.
2. 붙여넣고 싶은 곳으로 이동한다.
3. 내가 지정한 단축키를 누른다.
4. 숏컷이 클립보드에서 원하는 URL만 추출한다.
5. 추출한 URL만 자동으로 붙여넣는다.
```

예를 들어 클립보드에 아래 내용이 들어 있어도,

```text
아래 링크를 참고하세요.

    https://inf.run/Z6cNw
```

숏컷을 실행하면 아래 링크만 붙여넣어진다.

```text
https://inf.run/Z6cNw
```

## 왜 브라우저 확장이 아니라 macOS 숏컷인가?

이 문제는 크롬 확장 프로그램으로도 해결할 수 있다.
특정 페이지의 버튼 클릭을 감지하고, 복사된 클립보드를 다시 정리하는 방식이다.

하지만 처음부터 확장 프로그램을 만들면 설치, 권한, 브라우저별 관리까지 생각해야 한다.

반면 macOS 숏컷은 가볍다.

- 별도의 개발 환경이 필요 없다.
- macOS 기본 앱만으로 만들 수 있다.
- 크롬뿐 아니라 Safari, 메모장, Slack, Notion 등 어느 앱에서든 사용할 수 있다.
- 단축키 하나로 "클립보드 정리 + 붙여넣기"까지 처리할 수 있다.

이 문제는 브라우저 안에서 해결하기보다 **붙여넣기 직전에 클립보드를 정리하는 방식**으로 풀면 충분하다.

## 만들 숏컷의 핵심 로직

숏컷이 할 일은 단순하다.

```text
클립보드 가져오기
→ 정규식으로 원하는 URL 패턴 찾기
→ 찾은 URL만 클립보드에 다시 복사
→ Cmd + V 실행
```

이번 예시에서는 `https://inf.run/...` 형태의 짧은 URL만 추출해보겠다.

사용할 정규식은 다음과 같다.

```regex
https://inf\.run/[A-Za-z0-9]+
```

이 정규식은 아래와 같은 링크를 찾아낸다.

```text
https://inf.run/Z6cNw
https://inf.run/abc12
https://inf.run/A1B2C3
```

필요한 URL 형식이 다르다면 이 정규식만 바꾸면 된다.

## 1. 단축어 앱 열기

먼저 macOS에서 **단축어(Shortcuts)** 앱을 연다.

Spotlight에서 아래처럼 검색하면 바로 열 수 있다.

```text
단축어
```

영어 환경이라면 다음 이름으로 보일 수 있다.

```text
Shortcuts
```

앱을 열었다면 오른쪽 위의 `+` 버튼을 눌러 새 단축어를 만든다.

단축어 이름은 예를 들어 다음처럼 정하면 된다.

```text
URL만 붙여넣기
```

## 2. 클립보드 가져오기 액션 추가

첫 번째 액션으로 클립보드 내용을 가져온다.

검색창에서 다음 액션을 찾는다.

```text
클립보드 가져오기
```

영어 환경이라면 다음 액션이다.

```text
Get Clipboard
```

이 액션은 현재 클립보드에 들어 있는 텍스트를 다음 액션으로 전달한다.

## 3. 텍스트 일치 액션으로 URL 찾기

다음으로 정규식을 이용해 클립보드 내용에서 원하는 URL만 찾는다.

검색창에서 다음 액션을 추가한다.

```text
텍스트 일치
```

영어 환경에서는 다음 이름이다.

```text
Match Text
```

그리고 패턴 입력칸에 아래 정규식을 넣는다.

```regex
https://inf\.run/[A-Za-z0-9]+
```

이 액션은 클립보드 텍스트 안에서 `https://inf.run/`으로 시작하는 링크를 찾아준다.

## 4. 첫 번째 결과만 가져오기

`텍스트 일치` 액션의 결과는 리스트 형태다.
보통 이런 클립보드 정리에서는 필요한 링크가 하나만 들어 있으므로 첫 번째 결과만 가져오면 된다.

검색창에서 다음 액션을 추가한다.

```text
목록에서 항목 가져오기
```

영어 환경에서는 다음 이름이다.

```text
Get Item from List
```

설정은 다음처럼 맞춘다.

```text
첫 번째 항목 가져오기
```

영어 환경에서는 다음처럼 보일 수 있다.

```text
Get First Item
```

## 5. 추출한 URL을 클립보드에 복사하기

이제 정규식으로 찾은 첫 번째 URL을 다시 클립보드에 넣는다.

검색창에서 다음 액션을 추가한다.

```text
클립보드에 복사
```

영어 환경에서는 다음 이름이다.

```text
Copy to Clipboard
```

이 단계까지 완료하면 클립보드에는 긴 문장이 아니라 아래 URL만 남게 된다.

```text
https://inf.run/Z6cNw
```

## 6. 자동으로 붙여넣기

여기까지만 해도 "클립보드 정리" 숏컷으로는 충분하다.
하지만 목표는 단축키를 누르면 정리된 URL을 바로 붙여넣는 것이므로, 마지막에 `Cmd + V`를 실행하도록 AppleScript를 추가한다.

검색창에서 다음 액션을 추가한다.

```text
AppleScript 실행
```

영어 환경에서는 다음 이름이다.

```text
Run AppleScript
```

그리고 아래 코드를 입력한다.

```applescript
tell application "System Events"
  keystroke "v" using command down
end tell
```

이 AppleScript는 현재 활성화된 앱에 `Cmd + V` 키 입력을 보낸다.

즉, Slack 입력창에 커서가 있다면 Slack에 붙여넣고, 블로그 편집기에 커서가 있다면 블로그 편집기에 붙여넣는다.

## 7. 전체 액션 구성

최종 숏컷 구성은 다음 순서가 된다.

```text
1. 클립보드 가져오기
2. 텍스트 일치
   패턴: https://inf\.run/[A-Za-z0-9]+
3. 목록에서 항목 가져오기
   첫 번째 항목
4. 클립보드에 복사
5. AppleScript 실행
   tell application "System Events"
     keystroke "v" using command down
   end tell
```

## 8. 단축키 지정하기

이제 이 숏컷을 빠르게 실행할 단축키를 지정한다.

단축어 편집 화면에서 세부 정보 버튼을 누른 뒤, 키보드 단축키를 추가한다.

개인적으로는 기본 붙여넣기인 `Cmd + V`와 헷갈리지 않도록 아래처럼 조금 다른 조합을 추천한다.

```text
Control + Option + Command + V
```

또는 손에 더 익는 조합이 있다면 그것을 사용해도 된다.

중요한 점은 이 숏컷이 기본 붙여넣기를 완전히 대체하는 것이 아니라, **클립보드를 정리해서 붙여넣고 싶을 때 쓰는 별도 붙여넣기 단축키**라는 점이다.

## 9. 손쉬운 사용 권한 허용하기

마지막 단계의 AppleScript는 macOS에 키보드 입력을 보내는 동작을 한다.
그래서 처음 실행할 때 권한 요청이 뜰 수 있다.

권한이 필요하다고 나오면 다음 경로에서 단축어 앱 권한을 허용한다.

```text
시스템 설정
→ 개인정보 보호 및 보안
→ 손쉬운 사용
→ 단축어 허용
```

영어 환경에서는 대략 다음 경로다.

```text
System Settings
→ Privacy & Security
→ Accessibility
→ Shortcuts
```

권한을 허용한 뒤 다시 숏컷을 실행하면 자동 붙여넣기가 동작한다.

## 10. 사용해보기

웹페이지에서 필요한 URL이 포함된 텍스트를 복사한다.

클립보드에 아래처럼 들어왔다고 가정해보자.

```text
아래 링크를 참고하세요.

    https://inf.run/Z6cNw
```

이제 붙여넣고 싶은 곳에 커서를 둔 뒤, 앞에서 지정한 단축키를 누른다.

예를 들어 `Control + Option + Command + V`를 지정했다면 이 키를 누른다.

그러면 아래 값만 붙여넣어진다.

```text
https://inf.run/Z6cNw
```

## 예외 상황 처리하기

위 방식은 클립보드에 `https://inf.run/...` 링크가 있다고 가정한다.

만약 클립보드에 해당 패턴의 URL이 없는 상태에서 실행하면, `목록에서 항목 가져오기` 단계에서 결과가 없어서 숏컷이 멈출 수 있다.

더 안정적으로 만들고 싶다면 `텍스트 일치` 다음에 조건문을 추가할 수 있다.

```text
텍스트 일치 결과가 있는 경우
→ 첫 번째 항목 가져오기
→ 클립보드에 복사
→ Cmd + V 실행

그렇지 않은 경우
→ 그냥 Cmd + V 실행
```

이렇게 구성하면 클립보드에 원하는 URL이 있을 때는 그 값만 정제해서 붙여넣고, 없을 때는 일반 붙여넣기처럼 동작한다.

좀 더 실전적으로는 아래 흐름을 추천한다.

```text
1. 클립보드 가져오기
2. 텍스트 일치
   패턴: https://inf\.run/[A-Za-z0-9]+
3. 만약 일치 항목이 있다면
   - 첫 번째 항목 가져오기
   - 클립보드에 복사
   - AppleScript로 Cmd + V 실행
4. 그렇지 않다면
   - AppleScript로 Cmd + V 실행
```

이렇게 해두면 평소에는 일반 붙여넣기처럼 사용할 수도 있고, 원하는 URL 패턴이 클립보드에 있을 때만 자동으로 정제된다.

## 정규식 확장하기

현재 정규식은 특정 짧은 URL 패턴만 대상으로 한다.

```regex
https://inf\.run/[A-Za-z0-9]+
```

만약 추후 다른 도메인도 함께 처리하고 싶다면 정규식을 확장할 수 있다.

예를 들어 `inf.run` 링크와 일반 웹 URL을 모두 잡고 싶다면 아래처럼 바꿀 수 있다.

```regex
https://(inf\.run/[A-Za-z0-9]+|www\.example\.com/[^\s]+)
```

다만 자동화는 처음부터 범위를 넓히는 것보다, 내가 반복해서 겪는 작은 불편 하나만 정확히 줄이는 편이 낫다.
처음에는 특정 URL 패턴 하나만 처리하고, 나중에 필요할 때 확장하면 된다.

## 마무리

작은 자동화지만 자주 반복되는 작업일수록 효과가 크다.

복사한 텍스트에서 필요한 URL만 남기기 위해 앞뒤 문장을 지우는 일은 사소해 보인다.
하지만 개인 노트나 업무 메모에 같은 작업을 반복하다 보면 생각보다 자주 흐름이 끊긴다.

macOS 단축어를 이용하면 별도의 프로그램을 만들지 않고도 이 흐름을 다음 한 번의 단축키로 줄일 수 있다.

```text
복사
→ 정제해서 붙여넣기
```

처음에는 `Control + Option + Command + V`처럼 별도 단축키로 사용해보고, 손에 익으면 자신에게 맞는 조합으로 바꾸면 된다.

맥북의 단축어 앱은 거창한 자동화보다 이런 작은 반복을 줄일 때 더 자주 쓰게 된다.
복사한 텍스트를 그대로 붙여넣는 대신, 붙여넣기 직전에 한 번 가공하는 것.
이 정도만 해도 손이 꽤 편해진다.
