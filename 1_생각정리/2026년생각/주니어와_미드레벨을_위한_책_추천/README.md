# 주니어 개발자분들께 추천하고 싶은 책들

새해 라이브때 주니어~미드레벨 개발자분들께 추천하는 책들을 알려달라는 요청을 받았다.  
책 추천은 예전에도 계속 했던 것 같긴한데, 절판된 책들도 있고, AI 환경까지 겹치니 더욱 고민이 되었다.
  
요즘 시대에도 과거의 책들이 도움이 되나? 란 생각은 있다.  
그래서 기술적인 책은 추천을 안하는게 좋지 않을까 생각한다.  
배우려면 AI도 있고, 인프런도 있으니 기술적인 부분들은 책 외에도 배울 수 있는게 많고 이젠 어떤 기술이 필수적인가에서 모호한게 많으니 제외하는게 좋을것 같다.  
  
해서 지금 막 입사하실 신입/주니어/미드레벨 개발자분이라면 무엇을 추천할까 다시금 생각해봤다.
