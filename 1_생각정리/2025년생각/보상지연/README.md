# 보상지연

"좋은 회사에 합격하면 그때 컨퍼런스에 나가서 발표를 해야지" 같은 형태다.  

이를테면 국내에는 꼭 대형기업에서 1,000만 MAU 트래픽 경험 이외에 주니어 개발자분들 혹은 아주 많은 제약이 있는 상태에서의 경험을 필요로 하는 컨퍼런스들도 많다.  

[젋은 개발자들이 발표하는 유스콘](https://frost-witch-afb.notion.site/YOUTHCON-24-49177aa1b828435f98046912c6debc78)이나 [AWS 한국 사용자 모임에서 운영하는 여러 소모임들](https://www.awskr.org/), 요즘 활발하게 공유되는 [타입스크립트 백엔드 모임](https://www.linkedin.com/company/typescript-backend-meetup/posts/?feedView=all) 등등 다양한 주제로 많다.
