# 버티컬 벗어나기

29cm가 오늘의집이 1등을 하고 있는 리빙 카테고리로 영역을 확장해서 부딪치고
오늘의 집은 숨고가 1등을 하고 있는 시공으로 영역을 확장해서 부딪치고 있다


최근 잇섭의 테슬라 FSD 영상을 시청했는데 조회수가 158만인걸 보고 깜짝 놀랬습니다.
[테슬라 자율주행 진짜 미쳤나? ㄷㄷ FSD 한국 공식 오픈 기념 무한 테스트](https://www.youtube.com/watch?v=6qyQqxPsz20)

자동차 관련된 여러 유튜브 채널보다 IT 유튜버인 잇섭의 테슬라 영상이 조회수가 훨씬 높았기 때문인데요.
이젠 자동차 전문 유튜버 채널이라고 해도, IT 유튜버들의 경쟁을 받아야하는 상황이 된 것이죠.

이외에도 29cm과 컬리가 리빙으로 영역을 확장하면서 오늘의 집과 부딪치고 있습니다.
오늘의집은 리빙 영역에서 시공으로 영역을 확장해서 숨고와 부딪치고 있습니다.
온라인 리빙 시장 재편…오늘의집·29CM·컬리 3파전
